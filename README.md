# chatrooms

> Chat application built with ReactJS, ExpressJS, Socket.IO and NodeJS.
> [Live demo](https://chatrooms-socket-io.netlify.app/)


### Setup (frontend & backend)

```
npm i
cd frontend
npm i
```

### Run application

```
npm run dev
```
